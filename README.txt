UPS 2023
Para la descarga oficial del sistema se tiene que acceder al repositorio
de GitHub
https://github.com/EduardoArce03/Prueba-U2
Se puede descargar el codigo en .zip
Una ves descargado se extraen los archivos comprimidos
Se abre el terminal de comandos en el directorio raiz del proyecto
Procedemos a ejecutar 'npm install' para instalar paqueteria del frameword
Una vez instalados los paquetes, se ejecuta 'ng serve -o'
Si se desea editar el codigo
Se abre un compilador de codigo compatible con Angular (Se recomienda Visual Studio Code)
Se abre el directorio raiz
Se instala la paqueteria y se abre el proyecto con 'ng serve -o'
